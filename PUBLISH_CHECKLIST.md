# Publish Checklist

- [ ] Run `node scripts/build-site.mjs`.
- [ ] Run `node scripts/export-github-dataset.mjs`.
- [ ] Review `manifest.json` and `reports/latest-weekly-report.md`.
- [ ] Commit the generated repository.
- [ ] Push to the public GitHub dataset repository.
- [ ] Submit or refresh the GitHub repository URL in the public launch kit.
- [ ] Share the weekly report link through media, vendor and community channels.
